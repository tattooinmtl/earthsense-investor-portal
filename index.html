Our Mission

To create and maintain a decentralized, non-governmental seismic network that empowers students and teachers to conduct real-time geophysical surveys using user-friendly, low-cost seismographs — all while building a global early warning system.

Phase 1: Prototype & Validate

Build mock-up website, prototype ESP32 + ADXL345 modules, stream real XML data. Secure initial revenue to launch Phase 2.
